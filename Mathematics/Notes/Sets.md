# Sets

The number of subsets for a set with n elements is 2^n.

The cardinality of the cartesian product between two sets A and B is |A| * |B|.

The ordered pairs of the cartesian product should start with the element of the set on the left.

The cartesian product R x R contains all the points of the cartesian plane.

The cartesian product R x R x R contains all the points of the 3D coordinate system.
